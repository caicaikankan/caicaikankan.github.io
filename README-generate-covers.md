# 文章配图生成工具

## 功能
调用字节跳动火山方舟的 Seedream 4.5 模型 API，为博客文章生成符合内容的配图。

## 使用方法

### 1. 安装依赖
```bash
npm install axios
```

### 2. 运行脚本
```bash
node generate-covers.js
```

## 配置说明

- **API_KEY**: 你的火山方舟 API 密钥
- **MODEL_NAME**: 使用的模型名称（Seedream-4.5）
- **articles**: 文章列表，包含标题、分类、描述和文件名

## 生成逻辑

1. 为每篇文章生成随机风格的提示词
2. 调用 Seedream API 生成图片
3. 将生成的图片保存到 `public/covers/` 目录

## 风格选项

脚本会随机选择以下风格之一：
- 简约现代风格，深色背景，科技感十足
- 复古像素风格，怀旧感，80年代编程氛围
- 未来主义风格，霓虹灯光，赛博朋克美学
- 极简抽象风格，几何图形，色彩渐变
- 手绘插画风格，温暖色调，故事感强

## 注意事项

- 确保 API 密钥有效，且有足够的免费配额
- 生成的图片格式为 SVG，尺寸为 1600x900
- 如果生成失败，脚本会自动重试
